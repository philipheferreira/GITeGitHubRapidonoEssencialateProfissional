Utilizar o comando git status para mostrar as funcionalidades e operações dentro do 
ambiente

git status

Ira mostrar a branch atual, se foi dado os comitts e estão atualizados, 
as mudanças que foram feitas e ainda não foram realizados os commits e os arquivos
que não foram atualizados
