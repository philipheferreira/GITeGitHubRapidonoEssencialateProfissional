O merge vai gerar conflito quando eu fizer um merge pra branch original e ja tiver 
feito commits tanto na branch que usei a master como base pra criar e tem commits novos
de partes da master que ja foram modificadas depois de criar essa branch com base nela.

Nesse cenario tenho 2 branchs, a branch conflito e a branch master. As duas tem commits dife
rentes e independentes. Ou seja, se eu usar o git merge vai ocasionar conflito
